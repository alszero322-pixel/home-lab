# HomeLab

## Overview
このリポジトリは継続学習・ポートフォリオ作成・資格取得に向けて、実践学習するためのプロジェクトです。

## Goal
学習・資格取得・ポートフォリオの３つの観点でGoalを設定します。

### 学習目標
- 実践的にネットワークやクラウドなどをを学ぶ。
- 資格学習で得た知識をホームラボで扱う。

### 資格目標
- CCNP (Now)
- LPIC-1 (Now)
- LPIC-2
- 情報処置安全確保支援士
- AWS SAA / AWS Security Specialty
- CKA
- CKS
- OSCP

### ポートフォリオ
- 学習内容やトラブル内容を記録する。
- HomeLab Managerを開発する。
