# 雛形HomeLab

## Overview
このリポジトリは継続学習・ポートフォリオ作成・資格取得に向けて、実践学習するためのプロジェクトです。

## Goal
学習・資格取得・ポートフォリオの３つの観点でGoalを設定します。

### 学習目標
- 実践的にネットワークを学ぶ。

### 資格目標
- CCNP (Now)
- LPIC-1 (Now)
- LPIC-2
- セキスぺ
- AWS SAA / Security Supecialty
- CKA
- CKS
- OSCP

### ポートフォリオ
- 学習内容やトラブル内容を記録する。
- HomeLab Manager開発する。
